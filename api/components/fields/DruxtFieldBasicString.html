# DruxtFieldBasicString

Basic String field.

This component is intended to be rendered by the <druxt-field /> component.

See: DruxtField
Example

A wholesome pasta bake is the ultimate comfort food. This delicious bake is super quick to prepare and an ideal midweek meal for all the family.
<druxt-field
  data="A wholesome pasta bake is the ultimate comfort food. This delicious bake is super quick to prepare and an ideal midweek meal for all the family."
  :schema="{
    id: 'field_summary',
    type: 'basic_string'
  }"
/>
