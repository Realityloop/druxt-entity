# DruxtEntityContextMixin

Deprecated

Provides a context passthrough mechanism for Vue.js components.

Context persists through all <druxt-entity> and <druxt-field> components.

Kind: global mixin
Todo

  • Move DruxtEntityContextMixin to a common module.

Example

<!-- Parent component -->
<template>
  <child-component :context="{ icon: true }" />
</template>
